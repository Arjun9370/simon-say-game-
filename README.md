# All info about  Simon Game

A classic **Simon Memory Game** built using **HTML, CSS, and JavaScript**.  
The game is fully **mobile-friendly** and works on both **desktop and smartphones** using a **Start / Restart button** (no keyboard required).

##  Features
-  Works on mobile & desktop
- Start / Restart button
- 4 colorful buttons (Red, Yellow, Green, Purple)
- Memory-based gameplay
-  Smooth flash animations
-  Game-over feedback with score
-  Clean and simple UI
-  
## How to Play
1. Click the **Start Game** button  
2. Watch the color that flashes  
3. Repeat the sequence by clicking the same colors  
4. Each level adds one new color  
5. Make a mistake → Game Over   
6. Press **Start** again to play from Level 1

## Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling & animations  
- **JavaScript (Vanilla)** – Game logic
